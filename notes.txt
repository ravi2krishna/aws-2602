    -> Cloud Offerings 

        -> Cloud offerings provide on-demand access to 
        computing, storage, and software resources over the internet.

            -> IaaS (Infrastructure as a Service)

            -> PaaS (Platform as a Service)

            -> SaaS (Software as a Service)

        -> NOTE: Similar to Online Ride Offerings(Rapido's) 
        Scooty, Bikes, Autos, Hatchback, Sedan, SUV etc 

        
        -> IaaS (Infrastructure as a Service)

            -> Provides raw computing infrastructure virtual machines, servers, storage, and networking. 
            You manage the operating system and applications.

            -> https://miro.medium.com/v2/resize:fit:1400/1*VIzchB0D3t1awTzj3sfe1g.png

            -> Server For Myntra Application 

                -> Hardware will be provided and managed by vendor 

                    -> 4 CPU's - 16 GB RAM - 500 GB SSD - Networking 
                
                -> Customer(You) need to manage all below 
                    
                    -> Linux Ubuntu Operating System > Tomcat > Java Runtime > Database Server (MySQL) > Myntra App 

                -> Examples: AWS EC2 - AWS VPC - AWS EBS 


        -> PaaS (Platform as a Service)     

            -> Delivers a framework for "developers" to build, test, and 
            deploy software without managing the underlying hardware.

            -> Server For Myntra Application 

                -> Hardware & Platform will be provided and managed by vendor 

                        -> 4 CPU's - 16 GB RAM - 500 GB SSD - Networking - Linux Ubuntu Operating System > Tomcat > Java Runtime > Database Server (MySQL)
                    
                    -> Customer(You) need to manage all below 
                        
                        -> Myntra App 

                    -> Examples: AWS Elastic Beanstalk - AWS RDS - AWS EKS 


        -> SaaS (Software as a Service)

            ->  Fully functional, ready-to-use software applications 
            accessed via a web browser (e.g., Salesforce, Microsoft 365, Google Workspace).
                    
            -> Server For Myntra Application 

                -> Hardware & Platform & Software will be provided and managed by vendor 

                        -> 4 CPU's - 16 GB RAM - 500 GB SSD - Networking - Linux Ubuntu Operating System > Tomcat > Java Runtime > Database Server (MySQL) > Your Application 
                    
                    -> Customer(You) need to manage all below 
                        
                        -> NOTHING 

                    -> Examples: AWS Chime(Online Meetings, Video Conference) - 
                    AWS Workspaces(Desktop As Service) - AWS Connect(Contact Center Service)

        
        -> Customer-1: Needs Only Hardware i.e Only IaaS

        -> Customer-2: Needs Hardware + Platform i.e IaaS + PaaS 

        -> Customer-3: Needs Direct Application i.e SaaS 


    -> IaaS: Rent Infrastructure - You Manage OS, Middleware & Apps - AWS EC2, AWS VPC, AWS EBS

    -> PaaS: Build & Deploy Apps - You Manage Apps - AWS Elastic Beanstalk - AWS RDS, AWS EKS 

    -> SaaS: Use Apps - vendor manages everything - AWS Chime - AWS Connect - AWS Workspaces
